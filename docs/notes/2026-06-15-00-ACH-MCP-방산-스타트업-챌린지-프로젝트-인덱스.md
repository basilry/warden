# ACH MCP 방산 스타트업 챌린지 제출 준비

## 한 줄 요약

`ach-mcp`는 LLM을 판단자가 아니라 통역·오케스트레이터로 제한하고, 정보 분석 규율(ACH, SourceVet, 출처 신뢰도, 감사 로그, 판단 변경 통지)을 MCP 도구로 강제하는 프로토타입이다. 방향성 재정립 후 1차 제품은 **WARDEN 에이전트 하네스**로 둔다. MCP는 WARDEN이 능력을 연결하는 표준이고, ACH·SourceVet은 첫 번째 규율 모듈이다.

## 현재 코드 자산

- 로컬 프로젝트: `/Users/basilry/Projects/02018_ach_mcp`
- 핵심 문서: `README.md`, `docs/FRAMEWORK.md`, `docs/architecture.md`, `TRANSCRIPT.md`
- 핵심 코드:
  - `src/index.ts`: MCP 서버 인터페이스, Tools/Resources 등록
  - `src/core/engine.ts`: ACH 결정적 엔진
  - `src/sourcevet/engine.ts`: SourceVet 결정적 엔진
  - `demo/run-demo.ts`: 모델 없이 실행되는 심사용 데모
- 검증 완료:
  - `npm run build`
  - `npm run demo:auto`
  - `npm run demo -- --module=sourcevet --auto`

## 제출 포지셔닝

> 폐쇄망·망분리 환경에서도 운용 가능한 **통제형 AI 에이전트 + 규율 MCP 기반 방산 분석 보조 MVP**.  
> LLM은 통역과 툴 오케스트레이션만 맡고, 판단·출처 검증·감사·승인 게이트는 결정적 코드와 사람에게 둔다.
