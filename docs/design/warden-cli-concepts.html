WARDEN CLI Design Concepts

Claude/Gemini CLI의 장점은 유지하되, WARDEN은 분석 에이전트 서버를 실행하고 루프, 도구, 승인 상태를 신뢰감 있게 보여주는 방향으로 정리했습니다. 캐릭터성은 작은 멀린 모자 정도만 남기고, 실제 텍스트는 운영 콘솔에 맞는 명칭을 사용합니다.

Dark terminal Korean runtime logs Merlin hat only No fantasy labels
CONCEPT A

Sentinel Welcome Panel

Claude Code처럼 처음 실행 화면의 신뢰감을 주는 구성입니다. 좌측은 정체성, 우측은 시작 명령과 최근 실행을 분리합니다.

첫 실행 화면 Concept A는 `warden`만 입력했을 때의 웰컴 화면으로 적합합니다.
브랜드성 Concept B는 가장 기억에 남지만, 매 실행마다 쓰기에는 시각적 무게가 큽니다.
채팅 UX Concept D는 상시 대화 모드에 적합하고 구현 난이도도 낮습니다.
